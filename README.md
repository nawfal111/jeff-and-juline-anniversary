# 💛 Happy Anniversary — Jeffrey & Juline

A little celebration page made with love.

## How to open it

Just double-click `index.html` — that's it.

## Adding photos to the slideshow

1. Drop your photos (jpg / png / gif / webp) into the `images/` folder.
2. Run this once in Terminal:
   ```
   cd ~/Desktop/"jeff and juls" && ./update-images.sh
   ```
3. Refresh the page — your photos appear in the slideshow with cinematic
   Ken Burns animations and captions. ✨

(If you serve the folder with `python3 -m http.server` instead, photos are
detected automatically without running the script.)
